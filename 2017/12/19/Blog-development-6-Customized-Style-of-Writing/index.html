新文章创建：

  • 在根目录下，执行命令 hexo new "title"
  • 在 ~/blog/source/_posts/ 下新建 .md 结尾的文件来写新的文章

文章的模版文件

我们知道执行以下命令：

hexo new post "title 用来创建新文章

hexo new page "title 用来创建新菜单页面

hexo new draft "title 用来创建新草稿

在 Hexo 中执行 hexo new "title" 命令，默认是执行 hexo new post "title" 命令。若想修改默认设置，需在 站点配置文件 中修改，如下图所示：

执行 hexo new "title" 命令新建的文章，实际上是将 ~/blog/scaffolds/ 中的文章模板文件 post.md 复制到了 ~/blog/source/_posts/ 下。自定义此模板文件，可以使每次新建的文章都会有自定义的内容。

参考阅读 Hexo 官方文档的 Front-matter 和 页面变量。
